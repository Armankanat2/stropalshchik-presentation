# Наполнение слайда 95

Этот документ фиксирует рабочее наполнение финального слайда презентации по профессии `Стропальщик`.

Принцип:

- финальный слайд должен визуально рифмоваться с титульным;
- он не должен быть перегружен текстом;
- основной смысл: курс завершен, ключевой приоритет компании - безопасность.

## Слайд 95. Завершение презентации

- Основная мысль: обучение завершено, но главный вывод курса остается с работником в практике - безопасность всегда на первом месте.
- Основной визуал: `../assets/working-visuals/95/slide-95-final-background.jpeg`
- Визуальная логика: использовать ту же картинку, что и для первой страницы презентации, чтобы финал ощущался как смысловое и фирменное замыкание курса.
- Главный текст на слайде:
- `Спасибо за внимание`
- `Стропальщик`
- `Safety ferst`
- Дополнительная строка мелким кеглем:
- `Первое безопасность`
- Примечание для сборки: слайд лучше делать почти без дополнительных элементов; оставить фирменный фон, логотип `Велесстрой`, крупный завершающий текст и слоган как финальный акцент.

## Что уже можно отдавать в работу

- для финального слайда уже определен фон;
- текстовый состав минимальный и подходит для официального завершения курса;
- слайд `95` можно собирать сразу после тестового блока `80-94`.
