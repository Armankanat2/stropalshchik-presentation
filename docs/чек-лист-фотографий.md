# Чек-лист фотографий для продолжения работы

## Назначение документа

Этот чек-лист нужен для сбора недостающих фотографий, которые потребуются для наполнения презентации по профессии `Стропальщик`.

Правило использования:

- отмечать чекбокс после получения подходящей фотографии;
- если по одному пункту есть несколько удачных кадров, это лучше, чем один;
- приоритет отдавать реальным производственным фото без постановочности;
- если на фото есть люди, кадр должен быть пригоден для использования в учебных материалах компании.

## Приоритет 1. Критически важные фото

### 1. Подготовка крана к работе

- [ ] Кран установлен на все опоры.
- [ ] Крупный или средний план опор крана.
- [ ] Пример правильной установки крана на подготовленной площадке.
- [ ] Пример различия основания или грунта под краном.
- [ ] Работа крана вблизи котлована или траншеи.
- [ ] Наглядный кадр с безопасным расстоянием до бровки котлована.

### 2. Ограждение опасной зоны

- [ ] Опасная зона ограждена сигнальной лентой.
- [ ] Опасная зона ограждена стойками или переносными ограждениями.
- [ ] Видно, как организован проход людей вне опасной зоны.
- [ ] Есть предупреждающие знаки или таблички рядом с зоной работ.

### 3. Работа вблизи ЛЭП

- [ ] Общий вид крана рядом с ЛЭП.
- [ ] Наглядный кадр, где видно опасное соседство крана и проводов.
- [ ] Фото заземления крана, если оно доступно.
- [ ] Предупреждающие знаки или плакаты по электробезопасности.

### 4. Складирование грузов

- [ ] Правильное складирование труб.
- [ ] Правильное складирование металла.
- [ ] Правильное складирование ЖБИ.
- [ ] Правильное складирование тары или контейнеров.
- [ ] Видны проходы между штабелями и зона безопасного доступа.
- [ ] Пример неправильного складирования для слайда с ошибкой.

### 5. Аварийные и опасные ситуации

- [ ] Перекос груза.
- [ ] Неправильная строповка груза.
- [ ] Человек в опасной зоне рядом с грузом.
- [ ] Попытка удержать или поправить груз руками.
- [ ] Работа с поврежденным стропом или сомнительной оснасткой.

## Приоритет 2. Очень желательные фото

### 6. Работа стропальщика на площадке

- [ ] Стропальщик рядом с грузом в рабочей обстановке.
- [ ] Стропальщик подает команды крановщику.
- [ ] Совместная работа стропальщика и крановщика.
- [ ] Работа сигнальщика.
- [ ] Безопасное расположение работников относительно груза и крана.

### 7. Схемы строповки по типам грузов

- [ ] Строповка труб.
- [ ] Строповка металла.
- [ ] Строповка ЖБИ.
- [ ] Строповка контейнера.
- [ ] Строповка оборудования.
- [ ] Строповка длинномерного груза.

### 8. Подготовка груза к перемещению

- [ ] Осмотр груза перед подъемом.
- [ ] Проверка точек захвата или мест строповки.
- [ ] Подготовка маршрута перемещения.
- [ ] Пробный подъем груза.

## Приоритет 3. Полезные фото для усиления наглядности

### 9. Оснастка и маркировка

- [ ] Бирка текстильного стропа крупным планом.
- [ ] Бирка канатного стропа крупным планом.
- [ ] Бирка цепного стропа крупным планом.
- [ ] Шильда или маркировка на грузозахватном приспособлении.
- [ ] Маркировка на таре.

### 10. Дефекты и браковка

- [ ] Порезы или разрывы текстильного стропа.
- [ ] Повреждение канатного стропа.
- [ ] Повреждение цепного стропа.
- [ ] Дефекты грузозахватных приспособлений.
- [ ] Дефекты траверсы.
- [ ] Дефекты тары.

### 11. Производственная среда

- [ ] Рабочая площадка с краном и грузом общим планом.
- [ ] Проходы, зоны складирования и рабочие маршруты.
- [ ] Примеры хорошей организации рабочего места.
- [ ] Примеры плохой организации рабочего места.

## Требования к фотографиям

- [ ] Фото достаточно четкие, без сильного смаза.
- [ ] На фото хорошо виден главный объект съемки.
- [ ] Нет лишних отвлекающих деталей, если они мешают обучающему смыслу.
- [ ] Фото можно безопасно использовать во внутреннем учебном материале.
- [ ] На фото нет нежелательных водяных знаков.
- [ ] При необходимости можно обрезать фото без потери смысла.

## Что особенно полезно собрать в первую очередь

Если времени мало, сначала нужны именно эти фото:

- [ ] Кран на опорах.
- [ ] Площадка и расстояние до котлована.
- [ ] Ограждение опасной зоны.
- [ ] Работа рядом с ЛЭП.
- [ ] Правильное складирование грузов.
- [ ] Неправильная строповка или опасное действие.
- [ ] Реальная строповка труб, металла, ЖБИ или оборудования.

## Итог

Чек-лист считается достаточным для следующего этапа, если собраны:

- основные фото по безопасности;
- основные фото по строповке;
- основные фото по складированию;
- хотя бы несколько фото ошибок и опасных ситуаций;
- несколько крупных планов маркировки и дефектов.
