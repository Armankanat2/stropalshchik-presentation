# Список материалов для заполнения проекта

Дата обновления: `2026-05-04`

## Назначение

Этот список нужен для того, чтобы довести проект до полностью укомплектованного и юридически понятного состояния перед финальной передачей.

## Обязательные материалы от заказчика

- Подтверждение прав на материалы из `assets/исходные-презентации`.
- Подтверждение прав на материалы из `assets/исходные-пособия`.
- Подтверждение прав на материалы из `assets/исходные-фото`.
- Подтверждение прав или письменное согласование использования рабочих визуалов из `assets/working-visuals`.
- Утвержденный перечень тестовых вопросов и ответов для итогового блока `75-93`.
- Актуальный образец бланка ответов, если должен использоваться не текущий вариант из `assets/working-visuals/77-79/slide-79-answer-sheet.png`.

## Желательные исходники в лучшем качестве

- Оригинал или редактируемый источник схемы потери прочности стропа, которая сейчас используется на текущем `слайде 23`.
  Сейчас в сборке стоит растр: `assets/working-visuals/22-36/slide-23-strength-loss-chart.png`.
- Исходники или более качественные версии фото/схем для блока `24-34`.
  Особенно полезны оригиналы для стропов, грузозахватных приспособлений, траверс и тары.
- Исходники или более качественные версии схем для блока `50-57`.
  Это раздел со схемами строповки по типам грузов.
- Исходники или более качественные версии схем/фото для блока `58-74`.
  Это разделы по безопасности, аварийным ситуациям и итоговому повторению.

## Материалы по разделам

### Слайды 1-19

- Подтверждение, какие визуалы из раннего блока допустимо оставлять без замены.
- Если заказчик хочет полностью уйти от наследования старой версии, понадобятся новые фирменные фото для вводного блока.

### Слайды 20-34

- Фото и схемы по стропам.
- Фото и схемы по грузозахватным приспособлениям.
- Фото и схемы по траверсам.
- Фото и схемы по таре.
- При наличии: паспорта, шильды, бирки и маркировка в более читаемом качестве.

### Слайды 35-57

- Согласованные схемы алгоритма работ стропальщика.
- Согласованные схемы строповки по трубам, металлу, железобетону, контейнерам, оборудованию и длинномерным грузам.
- Если есть корпоративные учебные схемы в векторе или PDF, лучше заменить ими текущие растровые визуалы.

### Слайды 58-74

- Схемы по опасным зонам, ЛЭП, складированию, запретам и аварийным ситуациям.
- Если у заказчика есть утвержденные плакаты по охране труда, их стоит использовать как первоисточник для издательской версии.

### Слайды 75-93

- Финально утвержденные тестовые вопросы.
- Ключ правильных ответов для внутренней приемки.
- Если нужен отдельный печатный бланк, то его финальная версия в `PDF` или `DOCX`.
