# Карта источников для наполнения презентации

## Назначение документа

Этот документ нужен для того, чтобы быстро понимать:

- какой исходный материал является основным для каждого блока презентации;
- какие файлы лучше использовать для текста;
- какие файлы лучше использовать для фото, схем и примеров;
- где существующих материалов достаточно;
- где потребуется дополнительная подборка или генерация визуалов.

## Приоритет источников

### 1. Основа по содержанию

Основной источник для текста и учебной логики:

- `assets/исходные-пособия/Стропальные работы пособие/`
- `assets/исходные-пособия/ПОСОБИЕ ДЛЯ СТРОПАЛЬЩИКА.pdf`

Как использовать:

- брать оттуда базовые формулировки по профессии, порядку работ, требованиям безопасности, строповке и оснастке;
- использовать схемы и рисунки как методическую основу;
- упрощать язык под учебную аудиторию, а не переносить текст в слайды дословно.

Ограничение:

- пособие в папке представлено в виде сканов, поэтому это хороший источник смысла и иллюстраций, но не самый быстрый источник для копирования текста.

### 2. Основная презентация-исходник

Главный рабочий исходник для структуры и визуальной опоры:

- `assets/исходные-презентации/КУРС для СТРОПАЛЬЩИКА_III.pptx`
- резервная версия для просмотра: `assets/исходные-презентации/КУРС для СТРОПАЛЬЩИКА_III.pdf`

Как использовать:

- брать оттуда порядок раскрытия тем;
- использовать готовые блоки по стропам, грузозахватным приспособлениям, таре, браковке, схемам и безопасности;
- извлекать удачные таблицы, маркировки, примеры дефектов и наглядные слайды.

Что особенно полезно в этом файле:

- вступительные слайды по профессии и допуску;
- блоки по кранам и их параметрам;
- большой массив по стропам всех видов;
- блоки по захватам, траверсам и таре;
- темы складирования, безопасности и оценки рисков.

### 3. Практическая презентация

Вспомогательный, но очень полезный источник для прикладной части:

- `assets/исходные-презентации/КУРС для СТРОПАЛЬЩИКА_ПРАКТИКА.pptx`

Как использовать:

- брать оттуда формулировки для практического блока;
- использовать как опору для слайдов про сигналы, строповку, отбраковку и подготовку к работе;
- использовать для проверки, что теоретическая часть опирается на реальные действия, а не только на определения.

### 4. Тестовый блок

Основной источник для финальных вопросов:

- `assets/исходные-пособия/Итоговые тесты_Стропальщик_Русский.docx`

Как использовать:

- отобрать `15` вопросов для финального блока;
- переработать их под единый стиль презентации;
- сохранить один правильный ответ на каждый вопрос;
- часть вопросов можно сделать графическими, если в исходнике уже есть иллюстрации.

### 5. Бренд и фирменная подача

Основной источник фирменного оформления:

- `assets/бренд-материалы/логотип велесстрой.png`

Как использовать:

- брать логотип для всех слайдов;
- снимать фирменные цвета с логотипа;
- использовать как основу для плашки номера слайда, титула и фирменных акцентов.

### 6. Готовые фото и сигналы

Рабочие визуалы для ранних и повторяющихся блоков:

- `assets/исходные-фото/стоп.png`
- `assets/исходные-фото/поднять груз.png`
- `assets/исходные-фото/переместить стрелу..png`
- `assets/исходные-фото/внимание.png`
- `assets/исходные-фото/мало внимания.png`
- `assets/исходные-фото/Осторожно. Работник велесстрой кадр 1..png`

Как использовать:

- для знаковой сигнализации;
- для предупреждающих и закрепляющих слайдов;
- для вставок с практическими примерами;
- для визуального оживления вступительных и переходных слайдов.

## Карта источников по блокам презентации

### Слайды 1-4. Титул, оглавление, нормативка, структура обучения

Основные источники:

- `логотип велесстрой.png`
- `КУРС для СТРОПАЛЬЩИКА_III.pptx` — вступительные формулировки
- `техническое-задание.md`

Комментарий:

- эти слайды нужно собирать заново в фирменной подаче;
- существующие материалы здесь являются опорой по смыслу, а не готовой визуальной заготовкой.

### Слайды 5-10. Роль стропальщика и краткий блок по подъемным сооружениям

Основные источники:

- `КУРС для СТРОПАЛЬЩИКА_III.pptx`
- `ПОСОБИЕ ДЛЯ СТРОПАЛЬЩИКА.pdf`

Особенно полезны:

- слайды исходного курса с квалификационными требованиями;
- слайды про краны, параметры и опасные зоны.

Комментарий:

- этот блок можно собирать в основном из существующих материалов;
- часть схем лучше перерисовать компактнее, чтобы не перегружать начало презентации.

### Слайды 11-20. Допуск, начало работ, взаимодействие и связь

Основные источники:

- `КУРС для СТРОПАЛЬЩИКА_III.pptx`
- `ПОСОБИЕ ДЛЯ СТРОПАЛЬЩИКА.pdf`
- `техническое-задание.md`

Комментарий:

- по допуску и общим требованиям материалов достаточно;
- по иерархии, единому языку команд и распределению ролей лучше сделать новые понятные схемы специально под наш проект.

### Слайд 21. Основные знаковые сигналы стропальщика

Основные источники:

- `assets/исходные-фото/стоп.png`
- `assets/исходные-фото/поднять груз.png`
- `assets/исходные-фото/переместить стрелу..png`
- `КУРС для СТРОПАЛЬЩИКА_III.pptx`
- `КУРС для СТРОПАЛЬЩИКА_ПРАКТИКА.pptx`

Комментарий:

- этот слайд лучше собирать как новый компактный лист-шпаргалку;
- готовые изображения сигналов у нас уже есть, это ускоряет работу.

### Слайды 22-36. Стропы, грузозахватные приспособления, траверсы, тара

Основные источники:

- `КУРС для СТРОПАЛЬЩИКА_III.pptx`
- `Стропальные работы пособие/`
- `ПОСОБИЕ ДЛЯ СТРОПАЛЬЩИКА.pdf`
- `КУРС для СТРОПАЛЬЩИКА_ПРАКТИКА.pptx`

Что брать в первую очередь:

- из основного курса — маркировку, виды, браковку, типовые дефекты, правила применения;
- из практической презентации — прикладную формулировку и логику показа;
- из пособий — схемы и фото для усиления наглядности.

Комментарий:

- это самый хорошо обеспеченный исходниками блок;
- по нему можно начинать наполнение почти сразу.

### Слайды 37-48. Алгоритм выполнения работ стропальщика

Основные источники:

- `КУРС для СТРОПАЛЬЩИКА_III.pptx`
- `КУРС для СТРОПАЛЬЩИКА_ПРАКТИКА.pptx`
- `ПОСОБИЕ ДЛЯ СТРОПАЛЬЩИКА.pdf`

Особенно полезны:

- блоки исходного курса `Производство работ`;
- слайд по оценке рисков;
- практическая презентация по подготовке оснастки, груза и строповке.

Комментарий:

- по смыслу материалов достаточно;
- сам визуальный алгоритм лучше рисовать заново как понятную пошаговую схему.

### Слайды 49-59. Принципы строповки и схемы по типам грузов

Основные источники:

- `КУРС для СТРОПАЛЬЩИКА_III.pptx`
- `КУРС для СТРОПАЛЬЩИКА_ПРАКТИКА.pptx`
- `Стропальные работы пособие/`

Комментарий:

- для общих принципов и ошибок материалы есть;
- для отдельных типов грузов нужно проверить, хватает ли существующих схем именно под трубы, металл, ЖБИ, контейнеры, оборудование и длинномер;
- часть схем, скорее всего, придется добирать или рисовать заново в едином стиле.

### Слайды 60-68. Требования безопасности

Основные источники:

- `КУРС для СТРОПАЛЬЩИКА_III.pptx`
- `ПОСОБИЕ ДЛЯ СТРОПАЛЬЩИКА.pdf`
- `техническое-задание.md`

Хорошо покрыто:

- общие требования безопасности;
- складирование грузов;
- проверка оснастки;
- базовые ограничения при работе.

Потребует новых материалов:

- схема установки крана на опоры;
- наглядное объяснение разницы по грунтам;
- расстояние до котлована;
- таблица `зона разлета предметов`;
- наглядный слайд по ЛЭП с акцентом на заземление и правило `30 м`.

### Слайды 69-73. Аварийные ситуации и действия при нарушениях

Основные источники:

- `КУРС для СТРОПАЛЬЩИКА_III.pptx`
- `ПОСОБИЕ ДЛЯ СТРОПАЛЬЩИКА.pdf`

Комментарий:

- общую безопасность и логику остановки работ взять можно;
- полноценный наглядный аварийный кейс, скорее всего, нужно будет собрать заново;
- этот блок слабее обеспечен готовыми визуалами, чем блок по стропам и оснастке.

### Слайды 74-76. Итоговое повторение

Основные источники:

- `структура-презентации.md`
- ранее собранные слайды презентации
- готовые сигналы из `исходные-фото`

Комментарий:

- это не отдельный источник, а сборка на основе уже созданных материалов курса;
- новый контент здесь почти не нужен.

### Слайды 77-95. Аттестация, тесты, финал

Основные источники:

- `Итоговые тесты_Стропальщик_Русский.docx`
- изображения из `исходные-фото`
- отдельные картинки и схемы из исходных презентаций

Комментарий:

- тестовый блок можно собирать уже сейчас;
- часть вопросов удобно делать текстовыми, часть — по картинке;
- финальный слайд нужно собирать заново в фирменной подаче.

## Где существующих материалов достаточно

- блок по стропам;
- блок по грузозахватным приспособлениям;
- блок по таре;
- большая часть практического алгоритма работ;
- базовые темы по безопасности;
- тестовый блок.

## Где сразу нужны новые визуалы или генерация

- титульный слайд в фирменной подаче;
- оглавление и структура обучения;
- схема иерархии и взаимодействия на площадке;
- слайд про единый язык команд;
- компактная шпаргалка по знаковым сигналам;
- инфографика общего алгоритма работ;
- схема установки крана на опоры;
- схема безопасной площадки и расстояния до котлована;
- таблица `зона разлета предметов`;
- слайд по работе вблизи ЛЭП;
- часть схем строповки по конкретным типам грузов;
- кейс аварийной ситуации.

## Что использовать первым при наполнении

Рекомендуемый порядок работы:

1. Сначала наполнять `слайды 22-36`, потому что по ним лучше всего покрытие исходниками.
2. Затем брать `слайды 37-59`, потому что для алгоритма и строповки уже есть логика в основном и практическом курсах.
3. Потом собирать `слайды 60-68`, где часть материала есть, а часть уже понятно нужно дорисовывать.
4. После этого переходить к `слайдам 77-95`, потому что тесты уже лежат отдельным файлом.
5. В самом конце собирать самые фирменные и индивидуальные слайды: `1-4`, `13-21`, `69-76`.

## Вывод

Для быстрого старта лучше всего обеспечены материалами:

- блок `стропы, приспособления, траверсы, тара`;
- блок `алгоритм работ`;
- блок `тесты`.

Наибольшее количество новых визуалов потребуется для:

- фирменного вступления;
- схем взаимодействия;
- специальных условий безопасности;
- аварийных кейсов.
