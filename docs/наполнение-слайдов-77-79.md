# Наполнение слайдов 77-79

Этот документ фиксирует рабочее наполнение переходного блока перед тестовой частью.

Принцип:

- блок должен отделять учебную часть от аттестации;
- подача должна быть официальной, короткой и понятной;
- слушатель должен заранее понимать критерии сдачи и порядок заполнения бланка.

## Слайды 77-79. Переход к аттестации

### Слайд 77. Переход к итоговой проверке знаний

- Основная мысль: учебная часть завершена, далее следует итоговая проверка знаний по курсу.
- Тезисы на слайде:
- теоретическая часть курса завершена;
- далее проводится итоговая аттестация в форме тестирования;
- тестовый блок проверяет понимание ключевых правил и действий стропальщика;
- на этапе тестирования важно внимательно читать формулировки вопросов и ответов.
- Основной визуал: `../assets/working-visuals/77-79/slide-77-attestation-transition.png`
- Примечание для сборки: слайд лучше делать как спокойный официальный переход с крупным заголовком и 2-3 короткими пояснениями без перегруженной графики.

### Слайд 78. Критерии аттестации

- Основная мысль: условия прохождения итогового теста должны быть понятны до начала ответов.
- Тезисы на слайде:
- итоговая аттестация проводится в форме тестирования;
- тест включает `15` вопросов;
- к каждому вопросу предусмотрен один правильный ответ;
- успешная сдача теста: `12` правильных ответов и более;
- результат определяется по количеству верных ответов.
- Примечание для сборки: это служебный текстовый слайд, поэтому его лучше оформить как строгую таблицу или перечень правил без обязательного фонового изображения.

### Слайд 79. Инструкция по заполнению бланка ответов

- Основная мысль: слушатель должен одинаково понимать, как заполнять бланк, чтобы ошибки оформления не мешали проверке результата.
- Тезисы на слайде:
- перед началом заполнить `ФИО`, дату и подпись;
- ответы отмечать только в строках, соответствующих номеру вопроса;
- по каждому вопросу выбирать один вариант ответа;
- отметка должна быть четкой и однозначной;
- для текущего теста используются вопросы `1-15`.
- Основной визуал: `../assets/working-visuals/77-79/slide-79-answer-sheet.png`
- Примечание для сборки: лучше показать бланк крупно и добавить 3-4 короткие выноски к полям `ФИО`, `дата`, `подпись` и к ячейкам выбора ответов.

## Что уже можно отдавать в работу

- блок `77-79` уже можно собирать в презентации;
- `слайд 78` фиксирует утвержденный критерий прохождения: `12` правильных ответов и более;
- `слайд 79` уже опирается на реальный бланк из материалов по профессии `Стропальщик`.
