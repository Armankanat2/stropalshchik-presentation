# Наполнение слайдов 80-94

Этот документ фиксирует рабочее наполнение тестового блока по профессии `Стропальщик`.

Принцип:

- на каждом слайде размещается один вопрос;
- на каждом слайде должно быть `4` варианта ответа: `А`, `Б`, `В`, `Г`;
- формулировки берутся из утвержденного тестового файла и не усложняются;
- в сам слайд правильный ответ не выносится, он нужен только в ключе для проверки;
- вопросы в этом блоке отобраны только из тех тем, которые уже раскрыты в презентации.

## Общий формат тестовых слайдов

- заголовок: `Тестовый вопрос N`
- ниже: текст вопроса
- затем: `4` варианта ответа
- для вопросов по распознаванию изображений использовать готовые листы вариантов из папки `assets/working-visuals/80-94`
- стиль должен быть строгим и единообразным, без лишней декоративности

- Исходный вопрос: `Вопрос 1`
