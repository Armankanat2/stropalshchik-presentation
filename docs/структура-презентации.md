# Структура презентации

## Назначение документа

Этот документ фиксирует послайдовую структуру будущей презентации для передачи в издательство.

Документ нужен для:

- согласования логики презентации до начала верстки;
- фиксации содержания каждого слайда;
- контроля полноты разделов;
- поэтапного утверждения структуры.

## Общие правила для всех слайдов

- [ ] На каждом слайде должен быть логотип компании `Велесстрой`.
- [ ] На каждом слайде в верхней части должно быть указано название раздела.
- [ ] Все слайды должны иметь сквозную нумерацию.
- [ ] Номер слайда должен размещаться в нижнем правом углу.
- [ ] Номер слайда должен размещаться на брендированной плашке в фирменных цветах компании.
- [ ] Оформление должно быть выдержано в фирменных цветах компании `Велесстрой`.
- [ ] Слайды должны быть наглядными, с минимальным количеством текста.

## Раздел 1. Введение

### Слайд 1. Титульный лист

- [ ] Указать название компании: `Велесстрой`.
- [ ] Указать название профессии: `Стропальщик`.
- [ ] Оформить слайд в фирменных цветах компании.
- [ ] Разместить логотип `Велесстрой`.
- [ ] Сделать слайд визуально статусным и официальным, как стартовый титул обучающей программы.

Назначение слайда:

- сразу показать, для какой компании проводится обучение;
- обозначить тему и профессию;
- задать фирменный стиль всей презентации.

### Слайд 2. Оглавление

- [ ] Разместить оглавление презентации.
- [ ] Показать основные разделы обучения в логичной последовательности.
- [ ] Сделать оглавление простым и легко читаемым.

Назначение слайда:

- показать слушателю маршрут обучения;
- дать понимание структуры курса с самого начала.

### Слайд 3. Нормативная и законодательная база

- [ ] Указать, на какие законодательные и нормативные акты опирается обучение.
- [ ] Представить перечень кратко, без перегрузки текста.
- [ ] Оформить список так, чтобы он читался с экрана и не выглядел как сплошной текст.

Назначение слайда:

- показать правовую основу обучения;
- подтвердить, что программа базируется на действующих требованиях.

### Слайд 4. Структура обучения

- [ ] Показать общую структуру обучения: `теория`, `практика`, `тест`.
- [ ] Визуально разделить три блока, чтобы они легко воспринимались.
- [ ] Сделать акцент на последовательности прохождения обучения.

Назначение слайда:

- объяснить, как будет построено обучение;
- заранее подготовить слушателя к формату прохождения программы.
