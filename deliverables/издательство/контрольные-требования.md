# Контрольные требования

Этот документ фиксирует требования к контрольной проверке финальной презентации перед передачей в работу и перед приемкой.

## Общие параметры

- Формат файла: `PPTX`
- Итоговое количество слайдов: `95`
- Рабочий размер слайда: `720 x 540`
- Язык презентации: русский
- Презентация должна открываться в `Microsoft PowerPoint` без потери изображений и текста

## Фирменное оформление

- Для рабочих слайдов используется фирменная верхняя шапка из исходной презентации `КУРС для СТРОПАЛЬЩИКА_III.pptx`
- Логотип `Велесстрой` должен сохраняться на новых слайдах `77-95`
- Брендированная плашка номера слайда должна быть сохранена из исходной презентации
- Нумерация новых слайдов должна идти последовательно: `77-95`

## Содержательные требования

- Слайды `77-79` оформляют переход к итоговой аттестации
- Слайды `80-94` содержат `15` тестовых вопросов
- На каждом тестовом слайде должен быть только один вопрос
- Для тестовых вопросов используется формат ответов `А`, `Б`, `В`, `Г`
- Слайд `95` завершает курс и использует визуал, связанный с титульной страницей

## Аттестация

- Итоговый тест содержит `15` вопросов
- К каждому вопросу предусмотрен один правильный ответ
- Условие успешной сдачи: `12` правильных ответов и более
- На слайде `79` должен использоваться реальный бланк ответов

## Что проверить перед выпуском

- Все изображения на слайдах `77-95` отображаются
- Заголовки слайдов `77-95` соответствуют рабочим документам проекта
- Текст на тестовых слайдах читается без наложений и обрезки
- Номер слайда виден и соответствует фактическому номеру
- Финальный слайд `95` содержит текст `Спасибо за внимание`, `Стропальщик`, `Safety ferst`, `Первое безопасность`

## Ограничение текущего пакета

- Проверка прав на часть изображений и материалов в этот этап не включена
